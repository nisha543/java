 swap number using temp variable
import java.util.Scanner;

public class SwapThreeNumbers {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.println("Enter three numbers separated by space: ");
        int num1 = scanner.nextInt();
        int num2 = scanner.nextInt();
        int num3 = scanner.nextInt();

        // Swapping using a temporary variable
        int temp;

        temp = num1;
        num1 = num2;
        num2 = num3;
        num3 = temp;

        System.out.println("After swapping:");
        System.out.println("Number 1: " + num1);
        System.out.println("Number 2: " + num2);
        System.out.println("Number 3: " + num3);

        scanner.close();
    }
}

what is jdk and jre and jvm ?
JRE is an acronym for Java Runtime Environment. It is also written as Java RTE. 
The Java Runtime Environment is a set of software tools which are used for developing Java applications. 
It is used to provide the runtime environment.

JVM (Java Virtual Machine) is an abstract machine. 
It is called a virtual machine because it doesn't physically exist. 
It is a specification that provides a runtime environment in which Java bytecode can be executed.
